# 数据传输过程
应用层：读取数据
传输层：封装成TCP/UDP包。包含一个包头存储源端口和目标端口。
网络层：封装成IP报文，包含源IP地址和目标IP地址。如果报文过大，超过mtu，会被分片。接收端会判断和组装分片。
链路层：添加头文件组成可以传输的数据帧。
