# 枚举
`enum example : 类型
`{
`   a=0,b=1,c=3
`}

枚举是一组整数合集，example不是命名但是可以当作类型名，可以直接使用a，b，c。默认值为0开始的整数。

# 引用
一种指针使用的变形
`int a = 5;
`int& ref = a;`
`ref = 10;`
其中ref是a的引用，int&是一个关键字。引用在建立时就需要定义，它不是一个变量，不可以更改引用对象。

`void log(int& value){
``	value++;
``}

`int main(){
`    int a = 0;
`    log(a);`
``}
上述代码是可以直接更改a的值的。

# 字符串
字符串的字面量存在只读区域中，使用指针定义时不应改变字符串：
`char* name = "zihao"
`name[1] = 'a';
以上的使用有误，当我们希望得到一个可以改变的字符串时应该使用字符数组：
`char name[] = "zihao"`

R,表示原始字符串：
