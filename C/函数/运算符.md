# 逗号运算符
逗号运算符把两个表达式连接成一个表达式，并保证最左边的表达式最先求值。逗号运算符通常在for循环头的表达式中用于包含更多的信息。整个逗号表达式的值是逗号右侧表达式的值。

# 条件运算符(? :)
`x = (y < 0) ? -y : y;
该语句的意思是“如果y小于0，那么x = -y;否则，x = y”
